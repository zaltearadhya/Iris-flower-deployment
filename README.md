# 🌸 Iris Flower Classification System | Machine Learning

## 📌 Problem Statement
The objective of this project is to build a machine learning model that can accurately classify iris flowers into different species based on their features such as sepal length, sepal width, petal length, and petal width.

---

## 🎯 Objectives
- Develop a supervised machine learning model for classification
- Perform exploratory data analysis (EDA) to understand feature relationships
- Compare multiple ML algorithms and select the best-performing model
- Evaluate model performance using standard metrics

---

## 📊 Data Overview
- Dataset: Iris Dataset
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Iris Setosa
  - Iris Versicolor
  - Iris Virginica

---

## 🧹 Data Preparation and Processing
- Handled dataset loading and structure using Pandas
- Checked for missing values and ensured data cleanliness
- Performed feature scaling (if applied)
- Split dataset into training and testing sets

---

## 🤖 Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier

---

## 📈 Model Evaluation
- Achieved ~95–98% accuracy across models
- Used evaluation metrics:
  - Confusion Matrix
  - Precision
  - Recall
  - F1-Score

---

## 📊 Visualizations
- Feature distribution plots
- Pair plots for relationship analysis
- Correlation heatmap

---

## 🔍 Key Insights
- Petal length and petal width are the most important features for classification
- KNN and Decision Tree performed slightly better than Logistic Regression
- The dataset is well-structured and easily separable
